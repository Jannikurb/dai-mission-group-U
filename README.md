# Identifying and Predicting Successful Prediction Markets on Polymarket

## Project Overview
This project investigates whether machine learning can identify distinct types of prediction markets on Polymarket and whether these market characteristics improve the prediction of trading volume. In addition, we analyze the causal effect of market rewards on trading volume.

## Data
The cleaned dataset is included in the repository. If a larger dataset exceeds GitHub's file size limit, it can be reproduced using the data collection notebook or downloaded from the provided link.
